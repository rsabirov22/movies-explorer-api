## movies-explorer-api
Бэкенд часть приложения Movies-explorer

## Технологии

При разработке использовались:
- Node.js
- Express
- MongoDB

## Запуск проекта

`mongod` — подключение к базе данных
`npm run dev` — запускает сервер с hot-reload
